# 🍰 Bakery Website

A modern and responsive bakery website built using **HTML, CSS, and JavaScript**. This project showcases a clean UI design with multiple sections like hero, menu, featured food, and footer.

## 🚀 Features

* Responsive design for different screen sizes
* Attractive landing (hero) section
* Featured food/products section
* About and services sections
* Newsletter subscription UI
* Clean and structured footer
* Smooth UI styling and layout using Flexbox

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## 📂 How to Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser

## 🎯 Purpose

This project is built for practicing frontend development and improving UI design skills.
